SlideShare PRO features for free

Roll out stratey, landing page design and development

SlideShare decided to axe its premium model and roll-out all the PRO features to all users. We need to strategies as how to roll this out and many users still had yearly and monthly subscription going on.

We planned to roll out these features in phased manner and chose to release features in time frame where there are minimum refunds to be made, I designed the landing page where users can subscribe to recieve the updates on features. Page was made fully responsive so that i can be accessible on mobiles as well as tablets. It was a last minute ask and i did it in One days short time frame. The campaign went very well and we had a overwhelming response from media and users. It was widely covered by social media and appriciated by the users.

Features of template:
  • Responsive design to make it accessable on all devices
  • Simple copy and big icons for easy access to feature details
  • Prominent Call to action
