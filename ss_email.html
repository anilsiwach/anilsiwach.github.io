SlideShare responsive email framework

unified email design and framework development

Task was to redesign SlideShare emails and make them responsive as we saw an increasing trend towards mobile devices. We were seeing a regular increase in number of mails been opened on mobile devices so decided to make our mails responsive so it can adapt to all the devices.

My role in this project was to visual design new mails, I went ahead with research on standards been followed by internet majors and got convinced on unified look and feel for all the SlideShare mails which was missing intially, I made the first cut for the mails and forked zurb's Ink framework to base SlideShare email framework on same. Design was done in modules so that any one can create mails picking and mixing modules required and need not to bother designers to get it done which eventually saves a lot of valuable time.

Features of template:
  • Responsive design with with graceful degradation
  • Mobile first approach
  • Reusable componets
  • Consistant design accross all mails
  • Simple copy and layout

How I went about it

Design all scenarios

having a wide range of mails i had to divide the content into common components, so i created possible scenarios and how they will look like in mail.
